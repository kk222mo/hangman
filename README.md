# Консольная игра "Виселица"

Простой сервер для игры в "виселицу", написанный на языке Go.
Для компиляции запустите
```
build.bat
```
Пример работы:
```
 ________
 |      |
( )     |     _ _ _ а т и _ 
\|/     |
 |      |
/       |
    ____|__
Использованные буквы: л п к т ф и н а 
Введите букву: н
Эту букву Вы уже называли :)))
Введите букву: д
Не угадали :(

```
